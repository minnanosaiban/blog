# 連載11: セグメント発進力

有報の事業セグメント別データから「どの事業が業績を牽引しているか」を可視化する分析ツールです。

連載記事: [セグメント発進力で「業績を牽引するセグメント」を見つける](https://minnanosaiban.github.io/hotline/blog/2026/05/21/11_segment_analysis/)

## ファイル

| ファイル | 種別 | 内容 |
|---|---|---|
| `11_segments_make_images.py` | Matplotlib チャート生成 | 記事用チャート画像（セグメントカバレッジ・加速/減速 Top10・主要4銘柄2年推移）を PNG 出力 |

## データについて

本リポジトリには **コードのみ** を公開しています。データは同梱していません。

- **有報 XBRL から変換した JSON**: 提供元の規約により再配布禁止です（`03_xbrl_json` で取得・変換）。

## 動作環境

`11_segments_make_images.py` は `C:\stock_analysis` 配下のユーティリティモジュールとデータに依存しています。別環境で動かす場合はデータ取得部をご自身の環境に合わせて書き換えてください。
