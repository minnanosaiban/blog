# 連載05 ― 株価チャートアプリ集

5分足・日足チャート、複数銘柄比較、決算パターン分析の Streamlit アプリ3本です。

## アプリ一覧

| フォルダ | 内容 |
|---|---|
| `01_5min_chart/` | 5分足ローソク＋騰落率テーブル（複数銘柄） |
| `02_multi_chart/` | 複数銘柄カードグリッド（90日チャート＋PER/PBR/配当） |
| `03_earnings_pattern/` | 決算パターングリッド（発表後の値動きを5分類） |

## セットアップ

```bash
cd 01_5min_chart          # または 02_multi_chart / 03_earnings_pattern
pip install -r requirements.txt
streamlit run app.py
```

初回起動時はメイン画面に手順が表示されます。

## データの取得方法

各アプリが参照するデータは生データのためリポジトリに含まれていません。
以下の手順でご自身の環境に用意してください。

### 1. 株価データ（yfinance）

アプリ内「⬛ データ取得」→「データ取得」を開き、銘柄コードを入力して
**「株価を取得」** を押してください。yfinance 経由で自動的に取得・保存されます。

- 日足: `data/prices/daily/{コード}.parquet`
- 5分足: `data/prices/5min/{コード}.parquet`（`01_5min_chart` と `03_earnings_pattern` のみ）

> **再配布制限**: yfinance が参照する Yahoo Finance のデータは利用規約により**再配布禁止**です。

---

### 2. 東証 銘柄一覧（data_j.xls）

TOPIX500 の規模区分フィルタに使用します。

1. 以下の JPX 公式ページを開く  
   <https://www.jpx.co.jp/markets/statistics-equities/misc/01.html>
2. 「東証上場銘柄一覧」の Excel ファイルをダウンロード
3. `data/master/data_j.xls` に保存

> **再配布制限**: 東証（JPX）が著作権を保有するデータのため**再配布禁止**です。

---

### 3. 銘柄短縮名（stocks.csv）

`data/master/stocks.csv` はリポジトリに同梱しています（著者作成・再配布可）。  
`コード` と `銘柄` の2列 CSV です。必要に応じて追記・編集してください。

---

### 4. 証券会社の指標データ（`02_multi_chart` のみ）

PER / PBR / 配当利回りの表示に使用します。  
対応する証券会社のサービスから以下の CSV を取得し、`data/` 直下に配置してください。

| ファイル名 | 内容 |
|---|---|
| `113_EPS.csv` | EPS実績 |
| `213_EPS.csv` | EPS予想 |
| `215_BPS.csv` | BPS予想 |
| `141_配当金.csv` | 配当金 |

指標データがない場合は RSI・25MA乖離率のみ表示されます（アプリは動作します）。

> **再配布制限**: 証券会社が提供するデータは利用規約により**再配布禁止**です。

---

### 5. 決算発表日時（`03_earnings_pattern` のみ）

発表後の値動きパターン分析に使用します。

アプリ内「⬛ データ取得」→「earnings.csv をアップロード」から
以下の形式の CSV をアップロードしてください。

```
date,time,code
2026-05-09,15:30,5020
2026-05-09,16:00,6758
```

| 列 | 内容 |
|---|---|
| `date` | 決算発表日（YYYY-MM-DD） |
| `time` | 発表時刻（HH:MM）。引け後は 15:30 など |
| `code` | 銘柄コード（4桁） |

決算発表日時は [TDnet（適時開示情報閲覧サービス）](https://www.release.tdnet.info/inbs/I_main_00.html) で確認できます。

> **再配布制限**: TDnet の開示データは提供元の規約により**再配布禁止**です。
